# Simple Calculator

A clean and functional calculator web application built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clear display functionality
- Responsive design
- Simple and intuitive user interface

## How to Use

1. Open `index.html` in your web browser
2. Use the numeric keypad to input numbers
3. Click on operators (+, -, \*, /) to perform calculations
4. Press '=' to see the result
5. Use 'C' to clear the display

## Project Structure

```
├── index.html    # Main HTML file containing the calculator structure
├── style.css     # CSS file for styling the calculator
├── script.js     # JavaScript file containing calculator logic
└── README.md     # Project documentation
```

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Setup

No special setup or dependencies are required. Simply clone the repository and open `index.html` in a web browser.

```bash
git clone https://github.com/Greycode009/Simple-Calculator.git
cd CALC
```

Then open `index.html` in your preferred web browser.
